# Zotero Collection Structure Manager Plugin

![Collection Structure Manager logo](icons/icon-128.png)

一个独立的 Zotero 插件，用于保守地检查和管理 collection / 文件夹结构。

## 功能

- 在 Zotero `工具` 菜单中添加 `文件夹结构管理器...`
- 支持快捷键 `Shift + Cmd + M` 快速打开管理器；也兼容 `Cmd/Ctrl + Shift + J`
- 管理器窗口支持拖动标题栏移动位置，并可拖动右下角调整大小
- 支持从窗口四角调整大小，包括右上角
- 支持拖动中间分割线调整文件夹列表和操作面板宽度
- 以紧凑单行树形结构列出所有 collection，可逐级展开/折叠根文件夹和子文件夹
- 左侧文件夹树使用更紧凑行距，提高一屏可见文件夹数量
- 支持多选文件夹：单击单选，`Cmd/Ctrl + 单击` 增减选择，`Shift + 单击` 连续选择
- 支持明确设置源文件夹和目标文件夹，移动/合并不再依赖隐含的当前选中状态
- 支持记住窗口大小、位置和中间分割线宽度
- 支持在管理器窗口内用 `Cmd/Ctrl + +`、`Cmd/Ctrl + -` 调整字体大小，`Cmd/Ctrl + 0` 恢复默认
- 左侧文件夹行支持右键菜单，可快速设为源/目标、跳转、复制路径、收藏/取消收藏、展开/折叠
- 右侧操作按钮按选择、跳转、结构、修改分组排列
- 展开/折叠三角按钮更醒目
- 列出直接条目数、含子文件夹条目数、子文件夹数、重名提示
- 支持按文件夹名称或完整路径搜索
- 搜索时自动显示匹配文件夹及其上级路径，方便定位层级
- 搜索关键词会高亮显示
- 支持收藏文件夹、最近访问文件夹，并可按全部/收藏/最近/空文件夹筛选；多选时可批量收藏/取消收藏
- 支持扫描空文件夹，即直接条目、递归条目和子文件夹均为 0 的文件夹
- 支持复制 Markdown 或 CSV 格式的文件夹结构报告
- 支持最近操作日志，移动文件夹后可撤销上一次移动
- 支持跳转到选中文件夹，支持双击行或按回车跳转
- 支持复制选中文件夹完整路径
- 支持新建根文件夹
- 支持在选中文件夹下新建子文件夹
- 支持把一个或多个选中文件夹移动到根目录或另一个目标文件夹下
- 支持详细合并预览，并在输入源文件夹名称确认后把源文件夹合并到目标文件夹
- 所有结构修改操作都会二次确认

## 合并规则

- 把源文件夹中的直接条目加入目标文件夹
- 把源文件夹的直接子文件夹移动到目标文件夹下
- 最后删除源文件夹
- 禁止合并到自身或自己的子文件夹
- 合并不会删除文献条目、附件或笔记

## 移动/合并使用方式

1. 在左侧选择一个或多个文件夹
2. 点击 `设为源文件夹`
3. 选择目标文件夹后点击 `设为目标文件夹`，也可以在目标下拉框中选择目标位置
4. 点击 `移动到目标位置` 或 `预览合并`

说明：移动可以把目标设为根目录；合并必须使用一个具体目标文件夹，且源文件夹只能有一个。

## 构建 XPI

在插件目录运行：

```bash
bash build.sh
```

构建产物会出现在：

- `dist/zotero-collection-structure-manager-plugin-0.6.1.xpi`

## 安装

1. 在 Zotero 中打开 `工具 -> 插件`
2. 点击右上角齿轮
3. 选择 `Install Add-on From File...`
4. 选择构建好的 `.xpi`
5. 重启 Zotero

## 兼容性

- Zotero 7

当前 manifest 明确标记为兼容 Zotero `7.0.*`。

## 当前行为说明

- 当前版本已整合 `zotero-collection-quick-jumper-plugin` 的主要快速跳转能力
- 不修改文献条目内容、附件、笔记或标签
- 移动和合并会修改 collection 结构，请确认预览后再执行
- 不提供单独删除文件夹功能，避免误删结构
- 收藏和最近访问只保存到插件本地偏好中
