# Zotero Collection Structure Manager Plugin

![Collection Structure Manager logo](icons/icon-128.png)

一个用于管理 Zotero collection / 文件夹结构的独立插件。它把文件夹快速跳转、结构查看、收藏、搜索、条目数量统计、新建、移动和合并等常用操作集中到一个管理器窗口中，适合文献条目很多、文件夹层级很深的 Zotero 文库。

## 主要功能

- 快速查看全部文件夹结构，支持根文件夹和子文件夹逐级展开/折叠
- 搜索文件夹名称或完整路径，并高亮关键词
- 跳转到选中文件夹，支持双击、回车和右键菜单
- 收藏常用文件夹，查看最近访问文件夹，筛选空文件夹
- 显示直接条目数、含子文件夹条目数、子文件夹数和重名提示
- 复制文件夹完整路径，导出 Markdown 或 CSV 格式的结构报告
- 新建根文件夹或子文件夹
- 将一个或多个源文件夹移动到根目录或指定目标文件夹
- 预览并确认后合并文件夹，合并不会删除文献条目、附件或笔记

## 快速使用

安装并重启 Zotero 后，在 `工具` 菜单中点击 `文件夹结构管理器...` 打开窗口。

也可以使用快捷键：

- macOS：`Shift + Cmd + M`
- 兼容快捷键：`Cmd/Ctrl + Shift + J`

常用操作：

1. 在搜索框输入文件夹名称或路径，快速定位目标文件夹
2. 单击选中文件夹，双击或按回车跳转到该文件夹
3. 右键某一行，可快速设为源/目标、跳转、复制路径、收藏/取消收藏、展开/折叠
4. 多选文件夹后，可批量收藏、移动或作为合并源使用

## 移动与合并

移动文件夹：

1. 在左侧选择一个或多个文件夹
2. 点击 `设为源文件夹`
3. 选择目标文件夹后点击 `设为目标文件夹`，或在目标下拉框中选择目标位置
4. 点击 `移动到目标位置`

合并文件夹：

1. 选择一个源文件夹并点击 `设为源文件夹`
2. 选择一个具体目标文件夹并点击 `设为目标文件夹`
3. 点击 `预览合并`
4. 确认预览内容后，输入源文件夹名称执行合并

合并规则：

- 把源文件夹中的直接条目加入目标文件夹
- 把源文件夹的直接子文件夹移动到目标文件夹下
- 最后删除源文件夹本身
- 禁止合并到自身或自己的子文件夹
- 合并不会删除文献条目、附件、笔记或标签

## 细节与设置

- 管理器窗口支持拖动标题栏移动位置
- 支持从窗口四角调整大小，包括右上角
- 支持拖动中间分割线，调整左侧文件夹列表和右侧操作面板宽度
- 会记住窗口大小、位置和中间分割线宽度
- 左侧文件夹树使用紧凑单行布局，提高一屏可见文件夹数量
- 展开/折叠三角按钮更醒目，方便在大量层级中操作
- 支持多选文件夹：单击单选，`Cmd/Ctrl + 单击` 增减选择，`Shift + 单击` 连续选择
- 管理器窗口内可用 `Cmd/Ctrl + +`、`Cmd/Ctrl + -` 调整字体大小，`Cmd/Ctrl + 0` 恢复默认
- 右侧操作按钮按选择、跳转、结构、修改分组排列
- 搜索时会自动显示匹配文件夹及其上级路径，方便定位层级
- 空文件夹指直接条目、递归条目和子文件夹均为 0 的文件夹
- 支持最近操作日志，移动文件夹后可撤销上一次移动

## 当前行为说明

- 当前版本已整合 `zotero-collection-quick-jumper-plugin` 的主要快速跳转能力
- 不修改文献条目内容、附件、笔记或标签
- 移动和合并会修改 collection 结构，请确认预览后再执行
- 所有结构修改操作都会二次确认
- 不提供单独删除文件夹功能，避免误删结构
- 收藏和最近访问只保存到插件本地偏好中

## 安装

1. 在 Zotero 中打开 `工具 -> 插件`
2. 点击右上角齿轮
3. 选择 `Install Add-on From File...`
4. 选择构建好的 `.xpi`
5. 重启 Zotero

安装文件：

- `dist/zotero-collection-structure-manager-plugin-0.6.1.xpi`

## 构建 XPI

在插件目录运行：

```bash
bash build.sh
```

构建产物会出现在：

- `dist/zotero-collection-structure-manager-plugin-0.6.1.xpi`

## 兼容性

- Zotero 7

当前 manifest 明确标记为兼容 Zotero `7.0.*`。
